## Disable Auto Zoom for Form

set the font size of the form elements to at least 16px

```
@media screen and (-webkit-min-device-pixel-ratio:0) { 
  select:focus,
  textarea:focus,
  input:focus {
    font-size: 16px;
    background: #eee;
  }
}
```
